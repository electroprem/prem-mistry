PCB AGITATOR

•This is PCB Agitator which continuously agitates the copper clad board when it’s immersed in etching solution.
• This automates the process, and no mess is created, where in manual process one needs to look after when etching to agitate the board if it’s not done than oxide layer is formed which needs to be removed by exposing to air by agitating solution so that etching process gets faster.
• I have designed this by upcycling old CD Writer any programmed microcontroller for setting the speed for back and forward motion also added 2 indicator LEDs.
• For benchmarking I etched PCB using this and other by manual shaking under 1hr automated agitator PCB completed etching where other took more than 1hr to etch.
